Configuration for [Prettier](https://prettier.io/).

## Installation

```shell
yarn add @gunnarx2/prettier prettier@^2.1.2 -D
```

## Usage

Require `@gunnarx2/prettier` in the Prettier config.

```javascript
module.exports = require('@gunnarx2/prettier');
```
